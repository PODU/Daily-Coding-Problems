# Day 1228

## Difficulty

Easy

## Problem Statement

Given a list of possibly overlapping intervals, return a new list of intervals where all overlapping intervals have been merged.

The input list is not necessarily ordered in any way.

## Example

### Input
```
[(1, 3), (5, 8), (4, 10), (20, 25)]
```
### Output
```
[(1, 3), (4, 10), (20, 25)]
```

## Explanation

Merge any overlapping intervals in an unordered list and return the consolidated set of non-overlapping intervals.
