# Day 967

## Difficulty

Medium

## Problem Statement

You are given an array of length `n + 1` whose elements belong to the set `{1, 2, ..., n}`. By the pigeonhole principle, there must be a duplicate. Find it in linear time and space.

## Explanation

Find the guaranteed duplicate in an array of length n+1 containing values from 1 to n, using linear time and space.
