# Day 1501

## Difficulty

Easy

## Problem Statement

Given the root of a binary tree, find the most frequent subtree sum. The subtree sum of a node is the sum of all values under a node, including the node itself.

## Example

### Input
```
  5
 / \
2  -5
```
### Output
```
2
```

## Explanation

Compute every subtree sum and return the value that occurs most frequently; here 2 appears twice (the left leaf, and 2 + 5 - 5).
