# Day 1418

## Difficulty

Medium

## Problem Statement

Given a node in a binary search tree, return the next bigger element, also known
as the inorder successor.

You can assume each node has a parent pointer.

## Example

### Input
```
   10
  /  \
 5    30
     /  \
   22    35

node = 22
```
### Output
```
30
```

## Explanation

Given a node in a BST (with parent pointers), return its inorder successor, the next larger element.
