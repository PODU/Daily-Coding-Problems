# Day 378

## Difficulty

Medium

## Problem Statement

Write a function that takes in a number, string, list, or dictionary and returns its JSON encoding. It should also handle nulls.

## Example

### Input
```
[None, 123, ["a", "b"], {"c":"d"}]
```
### Output
```
'[null, 123, ["a", "b"], {"c": "d"}]'
```

## Explanation

Implement a JSON serializer that encodes numbers, strings, lists, dictionaries, and null values into a JSON-formatted string.
