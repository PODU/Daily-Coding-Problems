# Day 1595

## Difficulty

Hard

## Problem Statement

Given a string, split it into as few strings as possible such that each string is a palindrome.

For example, given the input string `racecarannakayak`, return `["racecar", "anna", "kayak"]`.

Given the input string `abc`, return `["a", "b", "c"]`.

## Example

### Input
```
racecarannakayak
abc
```
### Output
```
["racecar", "anna", "kayak"]
["a", "b", "c"]
```

## Explanation

Partition a string into the minimum number of contiguous substrings such that every substring is a palindrome.
