# Day 138

## Difficulty

Hard

## Problem Statement

Find the minimum number of coins required to make `n` cents.

You can use standard American denominations, that is, 1¢, 5¢, 10¢, and 25¢.

## Example

### Input
```
n = 16
```
### Output
```
3 (a 10¢, a 5¢, and a 1¢)
```

## Explanation

Using US coin denominations, compute the fewest coins needed to total n cents.
