# Day 1472

## Difficulty

Easy

## Problem Statement

Given a list of integers, return the largest product that can be made by multiplying any three integers.

You can assume the list has at least three integers.

## Example

### Input
```
[-10, -10, 5, 2]
```
### Output
```
500
```

## Explanation

Find the maximum product of any three integers in the list; for `[-10, -10, 5, 2]` it is `-10 * -10 * 5 = 500`.
