# Day 1560

## Difficulty

Medium

## Problem Statement

Given a set of closed intervals, find the smallest set of numbers that covers all the intervals. If there are multiple smallest sets, return any of them.

## Example

### Input
```
[0, 3], [2, 6], [3, 4], [6, 9]
```
### Output
```
{3, 6}
```

## Explanation

Find the minimum-size set of points such that every given interval contains at least one of the chosen points.
