# Day 1119

## Difficulty

Medium

## Problem Statement

Given an array of numbers `N` and an integer `k`, your task is to split `N` into `k` partitions such that the maximum sum of any partition is minimized. Return this sum.

## Example

### Input
```
N = [5, 1, 2, 7, 3, 4], k = 3
```
### Output
```
8
```

## Explanation

Split an array into k contiguous partitions so the largest partition sum is as small as possible, and return that minimized maximum sum.
