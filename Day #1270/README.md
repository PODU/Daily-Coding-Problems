# Day 1270

## Difficulty

Hard

## Problem Statement

Given a word W and a string S, find all starting indices in S which are anagrams of W.

## Example

### Input
```
W = "ab", S = "abxaba"
```
### Output
```
0, 3, 4
```

## Explanation

Find every starting index in string S where a substring is an anagram of the word W.
