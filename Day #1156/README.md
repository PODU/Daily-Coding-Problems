# Day 1156

## Difficulty

Easy

## Problem Statement

Print the nodes in a binary tree level-wise. For example, the following should print 1, 2, 3, 4, 5.

## Example

### Input
```
  1
 / \
2   3
   / \
  4   5
```
### Output
```
1, 2, 3, 4, 5
```

## Explanation

Print the nodes of a binary tree in level-order (breadth-first) traversal.
