# Day 649

## Difficulty

Easy

## Problem Statement

Given a string, return the first recurring character in it, or null if there is no recurring character.

## Example

### Input
```
"acbbac"
"abcdef"
```
### Output
```
"b"
null
```

## Explanation

Scan the string and return the first character that appears more than once, or null if all characters are unique.
