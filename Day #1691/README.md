# Day 1691

## Difficulty

Medium

## Problem Statement

Given an integer n, find the next biggest integer with the same number of 1-bits on.

## Example

### Input
```
6 (0110 in binary)
```
### Output
```
9 (1001 in binary)
```

## Explanation

Find the smallest integer larger than n that has the same number of set bits (1s) in its binary representation.
