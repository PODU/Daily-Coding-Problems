# Day 678

## Difficulty

Easy

## Problem Statement

Given an integer, find the next permutation of it in absolute order. For example, given `48975`, the next permutation would be `49578`.

## Example

### Input
```
48975
```
### Output
```
49578
```

## Explanation

Rearrange the digits of an integer to form the next greater number (next permutation in increasing order).
