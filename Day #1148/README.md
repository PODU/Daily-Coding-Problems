# Day 1148

## Difficulty

Easy

## Problem Statement

Given a linked list and a positive integer `k`, rotate the list to the right by `k` places.

## Example

### Input
```
7 -> 7 -> 3 -> 5 and k = 2
1 -> 2 -> 3 -> 4 -> 5 and k = 3
```
### Output
```
3 -> 5 -> 7 -> 7
3 -> 4 -> 5 -> 1 -> 2
```

## Explanation

Rotate a linked list to the right by k positions.
