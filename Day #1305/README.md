# Day 1305

## Difficulty

Hard

## Problem Statement

Write a program that computes the length of the longest common subsequence of three given strings.

## Example

### Input
```
"epidemiologist", "refrigeration", "supercalifragilisticexpialodocious"
```
### Output
```
5
```

## Explanation

Compute the length of the longest common subsequence shared by three strings; for the given example the LCS is "eieio" with length 5.
