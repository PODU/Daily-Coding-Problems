## Difficulty

## Problem Statement

Given a linked list and an integer `k`, remove the k-th node from the end of the list and return the head of the list.

`k` is guaranteed to be smaller than the length of the list.

Do this in one pass.

## Explanation

Remove the k-th node from the end of a linked list in a single pass and return the new head.
