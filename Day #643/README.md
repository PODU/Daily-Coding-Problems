# Day 643

## Difficulty

Hard

## Problem Statement

Write a program that computes the length of the longest common subsequence of three given strings.

## Example

### Input
```
"epidemiologist", "refrigeration", "supercalifragilisticexpialodocious"
```
### Output
```
5
```

## Explanation

Find the length of the longest subsequence common to all three strings; here it is 5, the subsequence "eieio".
