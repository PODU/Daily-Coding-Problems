# Day 821

## Difficulty

Easy

## Problem Statement

A fixed point in an array is an element whose value is equal to its index. Given a sorted array of distinct elements, return a fixed point, if one exists. Otherwise, return `False`.

## Example

### Input
```
[-6, 0, 2, 40]
[1, 5, 7, 8]
```
### Output
```
2
False
```

## Explanation

Find an element of a sorted, distinct-valued array whose value equals its index, or return False if none exists.
