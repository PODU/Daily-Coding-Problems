# Day 231

## Difficulty

Easy

## Problem Statement

Given a string with repeated characters, rearrange the string so that no two adjacent characters are the same. If this is not possible, return None.

## Example

### Input
```
"aaabbc", then "aaab"
```
### Output
```
"ababac", then None
```

## Explanation

Rearrange a string so no two adjacent characters are identical, returning None if no such arrangement exists.
