# Day 1607

## Difficulty

Easy

## Problem Statement

Determine whether there exists a one-to-one character mapping from one string `s1` to another `s2`.

## Example

### Input
```
s1 = abc, s2 = bcd
s1 = foo, s2 = bar
```
### Output
```
true
false
```

## Explanation

Return whether each character of s1 can be consistently mapped to a single character of s2 (a one-to-one mapping).
