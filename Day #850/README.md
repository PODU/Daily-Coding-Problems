# Day 850

## Difficulty

Hard

## Problem Statement

Given a set of characters `C` and an integer `k`, a De Bruijn sequence is a cyclic sequence in which every possible `k`-length string of characters in `C` occurs exactly once.

Create an algorithm that finds a De Bruijn sequence.

## Example

### Input
```
C = {0, 1}, k = 3
```
### Output
```
00010111
```

## Explanation

Construct a cyclic De Bruijn sequence so that every k-length string over the alphabet C appears exactly once; for C = {0, 1} and k = 3 one valid answer is 00010111.
