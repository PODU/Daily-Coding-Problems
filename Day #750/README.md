# Day 750

## Difficulty

Medium

## Problem Statement

Generate a finite, but an arbitrarily large binary tree quickly in O(1).

That is, `generate()` should return a tree whose size is unbounded but finite.

## Explanation

Design a generate() function that returns, in O(1) time, a binary tree of finite but unbounded (arbitrarily large) size, typically using lazy/on-demand node creation.
