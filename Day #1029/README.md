# Day 1029

## Difficulty

Hard

## Problem Statement

Find the minimum number of coins required to make n cents.

You can use standard American denominations, that is, 1¢, 5¢, 10¢, and 25¢.

## Example

### Input
```
n = 16
```
### Output
```
3 since we can make it with a 10¢, a 5¢, and a 1¢.
```

## Explanation

Find the minimum number of coins needed to total n cents using denominations of 1, 5, 10, and 25 cents.
