# Day 379

## Difficulty

Easy

## Problem Statement

Given a string, generate all possible subsequences of the string.

Note that `zx` is not a valid subsequence since it is not in the order of the given string.

## Example

### Input
```
xyz
```
### Output
```
x
y
z
xy
xz
yz
xyz
```

## Explanation

Generate every subsequence of the input string, preserving the original character order.
