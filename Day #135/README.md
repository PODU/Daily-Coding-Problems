# Day 135

## Difficulty

Easy

## Problem Statement

Given a binary tree, find a minimum path sum from root to a leaf.

## Example

### Input
```
  10
 /  \
5    5
 \     \
   2    1
       /
     -1
```
### Output
```
15 (path [10, 5, 1, -1])
```

## Explanation

Find the root-to-leaf path in a binary tree whose node values sum to the minimum possible total.
