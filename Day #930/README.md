# Day 930

## Difficulty

Medium

## Problem Statement

Given a linked list, sort it in O(n log n) time and constant space.

For example, the linked list `4 -> 1 -> -3 -> 99` should become `-3 -> 1 -> 4 -> 99`.

## Example

### Input
```
4 -> 1 -> -3 -> 99
```
### Output
```
-3 -> 1 -> 4 -> 99
```

## Explanation

Sort a linked list in O(n log n) time and constant extra space, typically via bottom-up merge sort.
