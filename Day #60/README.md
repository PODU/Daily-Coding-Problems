# Day 60

## Difficulty

Medium

## Problem Statement

Given a multiset of integers, return whether it can be partitioned into two subsets whose sums are the same.

## Example

### Input
```
{15, 5, 20, 10, 35, 15, 10}
{15, 5, 20, 10, 35}
```
### Output
```
{15, 5, 20, 10, 35, 15, 10} -> true (split into {15, 5, 10, 15, 10} and {20, 35}, which both add up to 55)
{15, 5, 20, 10, 35} -> false
```

## Explanation

Determine whether a multiset of integers can be split into two subsets with equal sums.
