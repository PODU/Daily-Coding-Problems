# Day 1093

## Difficulty

Medium

## Problem Statement

Given an array of integers, return a new array where each element in the new array is the number of smaller elements to the right of that element in the original input array.

## Example

### Input
```
[3, 4, 9, 6, 1]
```
### Output
```
[1, 1, 2, 1, 0]
```

## Explanation

For each element, count how many elements to its right are smaller, and return those counts as a new array.
