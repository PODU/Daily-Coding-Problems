# Day 494

## Difficulty

Medium

## Problem Statement

Given a circular array, compute its maximum subarray sum in O(n) time. A subarray can be empty, and in this case the sum is 0.

## Example

### Input
```
[8, -1, 3, 4]
[-4, 5, 1, 0]
```
### Output
```
15
6
```

## Explanation

Find the maximum subarray sum in a circular array in O(n) time; for [8, -1, 3, 4] the answer is 15 (numbers 3, 4, and 8 wrapping around), and for [-4, 5, 1, 0] the answer is 6 (numbers 5 and 1).
