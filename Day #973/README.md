# Day 973

## Difficulty

Medium

## Problem Statement

Given the mapping a = 1, b = 2, ... z = 26, and an encoded message, count the number of ways it can be decoded.

You can assume that the messages are decodable. For example, '001' is not allowed.

## Example

### Input
```
'111'
```
### Output
```
3
```

## Explanation

Count the number of distinct ways a digit string can be decoded into letters using the mapping a=1..z=26. For '111' the answer is 3 ('aaa', 'ka', 'ak').
