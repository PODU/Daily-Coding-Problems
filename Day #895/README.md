# Day 895

## Difficulty

Hard

## Problem Statement

Given a string which we can delete at most k, return whether you can make a palindrome.

## Example

### Input
```
'waterrfetawx', k = 2
```
### Output
```
True
```

## Explanation

Determine whether a string can be turned into a palindrome by deleting at most k characters; e.g. deleting f and x from 'waterrfetawx' gives 'waterretaw'.
