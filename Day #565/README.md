# Day 565

## Difficulty

Medium

## Problem Statement

Given an integer list where each number represents the number of hops you can make, determine whether you can reach to the last index starting at index 0.

## Example

### Input
```
[2, 0, 1, 0]
[1, 1, 0, 1]
```
### Output
```
True
False
```

## Explanation

Determine if you can reach the last index of a list starting from index 0, where each value is the maximum number of hops allowed from that position.
