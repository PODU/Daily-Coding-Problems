# Day 1387

## Difficulty

Easy

## Problem Statement

Given the root of a binary search tree, and a target `K`, return two nodes in the tree whose sum equals `K`.

## Example

### Input
```
K = 20

    10
   /   \
 5      15
       /  \
     11    15
```
### Output
```
Return the nodes 5 and 15.
```

## Explanation

In a binary search tree, find two nodes whose values add up to a given target K.
