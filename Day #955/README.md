# Day 955

## Difficulty

Easy

## Problem Statement

Suppose an arithmetic expression is given as a binary tree. Each leaf is an integer and each internal node is one of '+', '−', '∗', or '/'.

Given the root to such a tree, write a function to evaluate it.

## Example

### Input
```
    *
   / \
  +    +
 / \  / \
3  2  4  5
```
### Output
```
45
```

## Explanation

Evaluate an arithmetic expression represented as a binary tree where leaves are integers and internal nodes are operators; the example computes (3 + 2) * (4 + 5) = 45.
