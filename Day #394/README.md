# Day 394

## Difficulty

Easy

## Problem Statement

Given a binary tree and an integer k, return whether there exists a root-to-leaf path that sums up to `k`.

## Example

### Input
```
k = 18 and the following binary tree:

    8
   / \
  4   13
 / \   \
2   6   19
```
### Output
```
True
```

## Explanation

Determine whether any root-to-leaf path in the binary tree sums to k; here 8 -> 4 -> 6 sums to 18.
