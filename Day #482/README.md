# Day 482

## Difficulty

Medium

## Problem Statement

Given a binary search tree and a range `[a, b]` (inclusive), return the sum of the elements of the binary search tree within the range.

## Example

### Input
```
    5
   / \
  3   8
 / \ / \
2  4 6  10

range = [4, 9]
```
### Output
```
23 (5 + 4 + 6 + 8)
```

## Explanation

Given a BST and an inclusive range [a, b], sum the values of all nodes whose value falls within that range.
