# Day 1810

## Difficulty

Easy

## Problem Statement

Given an array of numbers and a number `k`, determine if there are three entries in the array which add up to the specified number `k`.

## Example

### Input
```
[20, 303, 3, 4, 25] and k = 49
```
### Output
```
true (20 + 4 + 25 = 49)
```

## Explanation

Determine whether any three elements of the array sum to k (the 3-sum decision problem).
