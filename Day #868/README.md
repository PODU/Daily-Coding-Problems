# Day 868

## Difficulty

Easy

## Problem Statement

Given a string, determine whether any permutation of it is a palindrome.

## Example

### Input
```
"carrace"
"daily"
```
### Output
```
true
false
```

## Explanation

Determine if any permutation of a string can form a palindrome, which holds when at most one character has an odd count.
