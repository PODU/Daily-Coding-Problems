# Day 1622

## Difficulty

Hard

## Problem Statement

Given an integer k and a string s, find the length of the longest substring that contains at most k distinct characters.

## Example

### Input
```
s = "abcba", k = 2
```
### Output
```
"bcb"
```

## Explanation

Find the length of the longest contiguous substring of s that uses no more than k distinct characters.
