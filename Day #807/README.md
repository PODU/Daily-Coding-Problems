# Day 807

## Difficulty

Hard

## Problem Statement

Given an integer k and a string s, find the length of the longest substring that contains at most k distinct characters.

## Example

### Input
```
s = "abcba", k = 2
```
### Output
```
3
```

## Explanation

Find the length of the longest substring of s containing no more than k distinct characters (here "bcb" has length 3).
