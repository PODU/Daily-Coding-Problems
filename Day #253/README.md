# Day 253

## Difficulty

Medium

## Problem Statement

Given a string and a number of lines `k`, print the string in zigzag form. In zigzag, characters are printed out diagonally from top left to bottom right until reaching the `k`th line, then back up to top right, and so on.

## Example

### Input
```
"thisisazigzag" and k = 4
```
### Output
```
t     a     g
 h   s z   a
  i i   i z
   s     g
```

## Explanation

Lay out the characters of a string in a zigzag pattern across k lines, descending then ascending diagonally, and print the result row by row.
