# Day 9

## Difficulty

Hard

## Problem Statement

Given a list of integers, write a function that returns the largest sum of non-adjacent numbers. Numbers can be `0` or negative.

Follow-up: Can you do this in O(N) time and constant space?

## Example

### Input
```
[2, 4, 6, 2, 5]
```
### Output
```
13
```

## Explanation

Find the maximum sum achievable by choosing elements such that no two are adjacent. For [2, 4, 6, 2, 5] pick 2, 6, and 5 for 13; for [5, 1, 1, 5] pick 5 and 5 for 10.
