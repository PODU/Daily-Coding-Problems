# Day 332

## Difficulty

Easy

## Problem Statement

Given integers `M` and `N`, write a program that counts how many positive integer pairs `(a, b)` satisfy the following conditions:

 * `a + b = M`
 * `a XOR b = N`

## Explanation

Count the number of ordered positive integer pairs (a, b) whose sum equals M and whose bitwise XOR equals N.
