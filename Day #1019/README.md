# Day 1019

## Difficulty

Easy

## Problem Statement

Given two rectangles on a 2D graph, return the area of their intersection. If the rectangles don't intersect, return 0.

## Example

### Input
```
{
    "top_left": (1, 4),
    "dimensions": (3, 3) # width, height
}

and

{
    "top_left": (0, 5),
    "dimensions": (4, 3) # width, height
}
```
### Output
```
6
```

## Explanation

Compute the overlapping area between two axis-aligned rectangles, returning 0 if they do not intersect.
