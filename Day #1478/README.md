# Day 1478

## Difficulty

Easy

## Problem Statement

Given the root of a binary tree, return a deepest node. For example, in the following tree, return d.

```
    a
   / \
  b   c
 /
d
```

## Example

### Input
```
    a
   / \
  b   c
 /
d
```
### Output
```
d
```

## Explanation

Find and return one of the deepest nodes (those at maximum depth) in a binary tree.
