# Day 863

## Difficulty

Medium

## Problem Statement

Generate a finite, but an arbitrarily large binary tree quickly in O(1).

That is, `generate()` should return a tree whose size is unbounded but finite.

## Explanation

Design generate() to produce a finite but arbitrarily large binary tree in O(1), typically by lazily expanding child nodes (with randomized termination) only when accessed.
