# Day 660

## Difficulty

Hard

## Problem Statement

You come across a dictionary of sorted words in a language you've never seen before. Write a program that returns the correct order of letters in this language.

## Example

### Input
```
['xww', 'wxyz', 'wxyw', 'ywx', 'ywz']
```
### Output
```
['x', 'z', 'w', 'y']
```

## Explanation

Given words sorted lexicographically in an unknown language, deduce the ordering of the alphabet in that language (alien dictionary / topological sort).
