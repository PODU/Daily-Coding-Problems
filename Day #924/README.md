# Day 924

## Difficulty

Easy

## Problem Statement

Given an array of integers out of order, determine the bounds of the smallest window that must be sorted in order for the entire array to be sorted.

## Example

### Input
```
[3, 7, 5, 6, 9]
```
### Output
```
(1, 3)
```

## Explanation

Find the smallest contiguous window whose sorting makes the whole array sorted; for [3, 7, 5, 6, 9] the bounds are (1, 3).
