# Day 1349

## Difficulty

Medium

## Problem Statement

Given the sequence of keys visited by a postorder traversal of a binary search tree, reconstruct the tree.

## Example

### Input
```
2, 4, 3, 8, 7, 5
```
### Output
```
    5
   / \
  3   7
 / \   \
2   4   8
```

## Explanation

Rebuild a binary search tree given only the postorder traversal of its keys.
