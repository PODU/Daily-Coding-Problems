## Difficulty

Easy

## Problem Statement

A fixed point in an array is an element whose value is equal to its index. Given a sorted array of distinct elements, return a fixed point, if one exists. Otherwise, return `False`.

## Example

### Input
```
[-6, 0, 2, 40]
[1, 5, 7, 8]
```
### Output
```
2
False
```

## Explanation

In a sorted array of distinct elements, find an index whose value equals the index itself (a fixed point), or return False if none exists.
