# Day 1590

## Difficulty

Medium

## Problem Statement

Given a string, find the length of the smallest window that contains every distinct character. Characters may appear more than once in the window.

## Example

### Input
```
"jiujitsu"
```
### Output
```
5, corresponding to the final five letters.
```

## Explanation

Find the length of the shortest contiguous substring that contains every distinct character present in the string.
