# Day 1203

## Difficulty

Hard

## Problem Statement

You come across a dictionary of sorted words in a language you've never seen before. Write a program that returns the correct order of letters in this language.

## Example

### Input
```
['xww', 'wxyz', 'wxyw', 'ywx', 'ywz']
```
### Output
```
['x', 'z', 'w', 'y']
```

## Explanation

Given a list of words sorted according to an unknown alphabet, derive the ordering of the letters in that language (an alien dictionary / topological sort problem).
