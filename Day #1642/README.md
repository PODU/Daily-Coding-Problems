# Day 1642

## Difficulty

Hard

## Problem Statement

Find an efficient algorithm to find the smallest distance (measured in number of words) between any two given words in a string.

## Example

### Input
```
words "hello" and "world", text "dog cat hello cat dog dog hello cat world"
```
### Output
```
1
```

## Explanation

Given two words and a body of text, find the minimum number of words separating any occurrence of the two words.
