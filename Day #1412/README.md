# Day 1412

## Difficulty

Medium

## Problem Statement

Generate a finite, but an arbitrarily large binary tree quickly in O(1).

That is, `generate()` should return a tree whose size is unbounded but finite.

## Explanation

Design a generate() function that returns a binary tree of finite but arbitrarily large (unbounded) size in O(1) time, typically using lazy evaluation.
