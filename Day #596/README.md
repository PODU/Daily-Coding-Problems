# Day 596

## Difficulty

Medium

## Problem Statement

Invert a binary tree.

## Example

### Input
```
    a
   / \
  b c
 / \ /
d e f
```
### Output
```
  a
 / \
 c b
 \ / \
  f e d
```

## Explanation

Mirror a binary tree by swapping the left and right children of every node.
