# Day 1255

## Difficulty

Easy

## Problem Statement

Given a list of numbers and a number `k`, return whether any two numbers from the list add up to `k`.

Bonus: Can you do this in one pass?

## Example

### Input
```
[10, 15, 3, 7], k = 17
```
### Output
```
true
```

## Explanation

Determine whether any two numbers in the list sum to k, ideally in a single pass.
