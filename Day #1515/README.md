# Day 1515

## Difficulty

Medium

## Problem Statement

Given a list of numbers, create an algorithm that arranges them in order to form the largest possible integer. For example, given `[10, 7, 76, 415]`, you should return `77641510`.

## Example

### Input
```
[10, 7, 76, 415]
```
### Output
```
77641510
```

## Explanation

Arrange a list of numbers in the order that concatenates them into the largest possible integer.
