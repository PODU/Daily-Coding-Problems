# Day 82

## Difficulty

Easy

## Problem Statement

Using a read7() method that returns 7 characters from a file, implement readN(n) which reads n characters.

## Example

### Input
```
File content: "Hello world", three read7() calls
```
### Output
```
"Hello w", "orld", ""
```

## Explanation

Implement a readN(n) function that returns exactly n characters from a file using only a provided read7() primitive, buffering leftover characters between calls.
