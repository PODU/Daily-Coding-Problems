# Day 937

## Difficulty

Hard

## Problem Statement

You're given a string consisting solely of (, ), and *. * can represent either a (, ), or an empty string. Determine whether the parentheses are balanced.

## Example

### Input
```
(()*
(*)
)*(
```
### Output
```
(()* and (*) are balanced. )*( is not balanced.
```

## Explanation

Given a string of (, ), and * characters where each * may stand for (, ), or an empty string, determine whether the parentheses can be balanced.
