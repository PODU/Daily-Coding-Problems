# Day 1804

## Difficulty

Medium

## Problem Statement

You are given an array of length `n + 1` whose elements belong to the set `{1, 2, ..., n}`. By the pigeonhole principle, there must be a duplicate. Find it in linear time and space.

## Explanation

Given an array of n+1 elements drawn from {1..n}, find the value that appears more than once, using linear time and space.
