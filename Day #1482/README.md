# Day 1482

## Difficulty

Easy

## Problem Statement

Write a program that checks whether an integer is a palindrome. For example, `121` is a palindrome, as well as `888`. `678` is not a palindrome. Do not convert the integer into a string.

## Example

### Input
```
121
888
678
```
### Output
```
121 -> palindrome
888 -> palindrome
678 -> not a palindrome
```

## Explanation

Determine whether an integer reads the same forwards and backwards using only numeric operations, without converting it to a string.
