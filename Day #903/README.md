# Day 903

## Difficulty

Easy

## Problem Statement

Given an array of numbers and a number `k`, determine if there are three entries in the array which add up to the specified number `k`.

## Example

### Input
```
[20, 303, 3, 4, 25], k = 49
```
### Output
```
true
```

## Explanation

Decide whether any three entries of the array sum to k (3-sum existence); e.g. 20 + 4 + 25 = 49.
