# Day 1261

## Difficulty

Hard

## Problem Statement

Given a list of words, find all pairs of unique indices such that the concatenation of the two words is a palindrome.

## Example

### Input
```
["code", "edoc", "da", "d"]
```
### Output
```
[(0, 1), (1, 0), (2, 3)]
```

## Explanation

Find every pair of distinct indices whose words, concatenated in order, form a palindrome.
