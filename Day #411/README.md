# Day 411

## Difficulty

Hard

## Problem Statement

Given a string which we can delete at most k, return whether you can make a palindrome.

For example, given 'waterrfetawx' and a k of 2, you could delete f and x to get 'waterretaw'.

## Example

### Input
```
s = 'waterrfetawx', k = 2
```
### Output
```
True
```

## Explanation

Determine whether a string can be turned into a palindrome by deleting at most k characters.
