# Day 841

## Difficulty

Easy

## Problem Statement

Given integers `M` and `N`, write a program that counts how many positive integer pairs `(a, b)` satisfy the following conditions:

 * `a + b = M`
 * `a XOR b = N`

## Explanation

Count the positive integer pairs (a, b) such that a + b equals M and a XOR b equals N.
