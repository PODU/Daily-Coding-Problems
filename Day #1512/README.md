# Day 1512

## Difficulty

Easy

## Problem Statement

Given a string, return the first recurring character in it, or null if there is no recurring character.

For example, given the string "acbbac", return "b". Given the string "abcdef", return null.

## Example

### Input
```
"acbbac"
```
### Output
```
"b"
```

## Explanation

Find the first character in a string that appears more than once (by its first repeated occurrence), or null if all characters are unique.
