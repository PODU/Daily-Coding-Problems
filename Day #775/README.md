# Day 775

## Difficulty

Easy

## Problem Statement

Given an array of time intervals (start, end) for classroom lectures (possibly overlapping), find the minimum number of rooms required.

## Example

### Input
```
[(30, 75), (0, 50), (60, 150)]
```
### Output
```
2
```

## Explanation

Find the maximum number of overlapping intervals, which equals the minimum number of rooms needed.
