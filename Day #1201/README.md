## Difficulty

Medium

## Problem Statement

Write an algorithm that computes the reversal of a directed graph. For example, if a graph consists of `A` -> `B` -> `C`, it should become `A <- B <- C`.

## Example

### Input
```
A -> B -> C
```
### Output
```
A <- B <- C
```

## Explanation

Reverse the direction of every edge in a directed graph.
