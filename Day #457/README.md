# Day 457

## Difficulty

Hard

## Problem Statement

Given a word W and a string S, find all starting indices in S which are anagrams of W.

For example, given that W is "ab", and S is "abxaba", return 0, 3, and 4.

## Example

### Input
```
W = "ab", S = "abxaba"
```
### Output
```
0, 3, 4
```

## Explanation

Find every starting index in S where a substring is an anagram of W.
