# Day 1646

## Difficulty

Medium

## Problem Statement

Write a function that rotates a list by k elements. Try solving this without creating a copy of the list. How many swap or move operations do you need?

## Example

### Input
```
[1, 2, 3, 4, 5, 6], k = 2
```
### Output
```
[3, 4, 5, 6, 1, 2]
```

## Explanation

Rotate the elements of a list to the left by k positions in place, without allocating a copy.
