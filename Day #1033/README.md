# Day 1033

## Difficulty

Hard

## Problem Statement

Given an array of positive integers, divide the array into two subsets such that the difference between the sum of the subsets is as small as possible.

## Example

### Input
```
[5, 10, 15, 20, 25]
```
### Output
```
{10, 25} and {5, 15, 20}, which has a difference of 5, which is the smallest possible difference.
```

## Explanation

Partition an array of positive integers into two subsets minimizing the absolute difference of their sums.
