# Day 942

## Difficulty

Medium

## Problem Statement

Given a string of parentheses, write a function to compute the minimum number of parentheses to be removed to make the string valid (i.e. each open parenthesis is eventually closed).

## Example

### Input
```
"()())()"
")("
```
### Output
```
1
2
```

## Explanation

Given a string of parentheses, compute the minimum number of parentheses that must be removed so that the remaining string is balanced.
