# Day 1025

## Difficulty

Easy

## Problem Statement

Given a linked list, remove all consecutive nodes that sum to zero. Print out the remaining nodes.

## Example

### Input
```
3 -> 4 -> -7 -> 5 -> -6 -> 6
```
### Output
```
5
```

## Explanation

Repeatedly remove consecutive runs of linked-list nodes whose values sum to zero, then output the remaining nodes.
