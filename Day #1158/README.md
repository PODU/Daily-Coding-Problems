# Day 1158

## Difficulty

Medium

## Problem Statement

Given a linked list, sort it in O(n log n) time and constant space.

## Example

### Input
```
4 -> 1 -> -3 -> 99
```
### Output
```
-3 -> 1 -> 4 -> 99
```

## Explanation

Sort a singly linked list in O(n log n) time using constant extra space.
