# Day 260

## Difficulty

Medium

## Problem Statement

The sequence `[0, 1, ..., N]` has been jumbled, and the only clue you have for its order is an array representing whether each number is larger or smaller than the last. Given this information, reconstruct an array that is consistent with it.

## Example

### Input
```
[None, +, +, -, +]
```
### Output
```
[1, 2, 3, 0, 4]
```

## Explanation

Given an array of +/- signs describing whether each element rises or falls relative to the previous one, rebuild any permutation of [0, 1, ..., N] consistent with those comparisons.
