# Day 1807

## Difficulty

Medium

## Problem Statement

Given an array of numbers `N` and an integer `k`, your task is to split `N` into `k` partitions such that the maximum sum of any partition is minimized. Return this sum.

## Example

### Input
```
N = [5, 1, 2, 7, 3, 4], k = 3
```
### Output
```
8 (optimal partition is [5, 1, 2], [7], [3, 4])
```

## Explanation

Partition the array into k contiguous groups so that the largest group-sum is as small as possible, and return that minimized maximum sum.
