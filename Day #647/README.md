# Day 647

## Difficulty

Medium

## Problem Statement

Given a multiset of integers, return whether it can be partitioned into two subsets whose sums are the same.

## Example

### Input
```
{15, 5, 20, 10, 35, 15, 10}
{15, 5, 20, 10, 35}
```
### Output
```
true
false
```

## Explanation

Determine if a multiset can be split into two subsets with equal sums; the first set splits into {15, 5, 10, 15, 10} and {20, 35} (both summing to 55), while the second cannot be split evenly.
