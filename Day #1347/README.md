# Day 1347

## Difficulty

Easy

## Problem Statement

Given a string with repeated characters, rearrange the string so that no two adjacent characters are the same. If this is not possible, return `None`.

## Example

### Input
```
"aaabbc"
```
### Output
```
"ababac"
```

## Explanation

Rearrange a string so no two adjacent characters match, or return None if impossible. For example "aaab" returns None.
