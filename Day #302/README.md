# Day 302

## Difficulty

Medium

## Problem Statement

You are given a 2-d matrix where each cell consists of either /, \, or an empty space. Write an algorithm that determines into how many regions the slashes divide the space.

## Example

### Input
```
\    /
 \  /
  \/
(a three-by-six grid)
```
### Output
```
3
(considering the edges of the matrix as boundaries, this divides the grid into three triangles)
```

## Explanation

Given a grid of forward/back slashes and blanks, count how many separate regions the slashes carve the space into.
