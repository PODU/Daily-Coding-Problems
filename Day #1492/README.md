# Day 1492

## Difficulty

Easy

## Problem Statement

Given an array of time intervals (start, end) for classroom lectures (possibly overlapping), find the minimum number of rooms required.

## Example

### Input
```
[(30, 75), (0, 50), (60, 150)]
```
### Output
```
2
```

## Explanation

Find the minimum number of rooms needed to schedule all lectures given their (possibly overlapping) time intervals.
