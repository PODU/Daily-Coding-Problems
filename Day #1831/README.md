# Day 1831

## Difficulty

Easy

## Problem Statement

Given a sorted array, find the smallest positive integer that is not the sum of a subset of the array.

Do this in `O(N)` time.

## Example

### Input
```
[1, 2, 3, 10]
```
### Output
```
7
```

## Explanation

Find the smallest positive integer that cannot be formed as the sum of any subset of the given sorted array, in linear time.
