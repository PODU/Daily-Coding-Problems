# Day 69

## Difficulty

Easy

## Problem Statement

Given a list of integers, return the largest product that can be made by multiplying any three integers.

You can assume the list has at least three integers.

## Example

### Input
```
[-10, -10, 5, 2]
```
### Output
```
500 (since that's -10 * -10 * 5)
```

## Explanation

Find the maximum product obtainable by multiplying any three integers from a list.
