# Day 766

## Difficulty

Medium

## Problem Statement

You are given a string consisting of the letters `x` and `y`, such as `xyxxxyxyy`. In addition, you have an operation called `flip`, which changes a single `x` to `y` or vice versa.

Determine how many times you would need to apply this operation to ensure that all `x`'s come before all `y`'s.

## Example

### Input
```
xyxxxyxyy
```
### Output
```
2
```

## Explanation

In the example it suffices to flip the second and sixth characters, so you should return 2. Find the minimum flips to sort the string into all x's followed by all y's.
