# Day 373

## Difficulty

Hard

## Problem Statement

Given a list of integers L, find the maximum length of a sequence of consecutive numbers that can be formed using elements from L.

## Example

### Input
```
L = [5, 2, 99, 3, 4, 1, 100]
```
### Output
```
5
```

## Explanation

Find the length of the longest run of consecutive integers (e.g. [1, 2, 3, 4, 5]) that can be built from the elements of the list.
