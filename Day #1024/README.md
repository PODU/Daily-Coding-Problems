# Day 1024

## Difficulty

Easy

## Problem Statement

Given a 32-bit integer, return the number with its bits reversed.

## Example

### Input
```
1111 0000 1111 0000 1111 0000 1111 0000
```
### Output
```
0000 1111 0000 1111 0000 1111 0000 1111
```

## Explanation

Reverse the order of all 32 bits in a given 32-bit integer and return the result.
