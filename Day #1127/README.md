# Day 1127

## Difficulty

Medium

## Problem Statement

Given a string, find the length of the smallest window that contains every distinct character. Characters may appear more than once in the window.

## Example

### Input
```
"jiujitsu"
```
### Output
```
5
```

## Explanation

Find the length of the smallest substring (window) that contains every distinct character present in the string.
