# Day 906

## Difficulty

Easy

## Problem Statement

Given a set of points (x, y) on a 2D cartesian plane, find the two closest points.

## Example

### Input
```
[(1, 1), (-1, -1), (3, 4), (6, 1), (-1, -6), (-4, -3)]
```
### Output
```
[(-1, -1), (1, 1)]
```

## Explanation

Find the pair of points with the smallest Euclidean distance among a set of 2D points (closest pair of points).
