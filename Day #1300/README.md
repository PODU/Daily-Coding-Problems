# Day 1300

## Difficulty

Medium

## Problem Statement

Given a list of integers and a number K, return which contiguous elements of the list sum to K.

## Example

### Input
```
[1, 2, 3, 4, 5], K = 9
```
### Output
```
[2, 3, 4]
```

## Explanation

Find a contiguous subarray of the given list whose elements add up to K and return that subarray.
