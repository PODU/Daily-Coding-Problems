# Day 1303

## Difficulty

Medium

## Problem Statement

Given an integer n, find the next biggest integer with the same number of 1-bits on.

## Example

### Input
```
6 (0110 in binary)
```
### Output
```
9 (1001)
```

## Explanation

Given an integer, find the smallest integer larger than it that has the same number of set bits (1-bits).
