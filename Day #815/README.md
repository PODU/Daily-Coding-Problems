# Day 815

## Difficulty

Medium

## Problem Statement

The area of a circle is defined as πr^2. Estimate π to 3 decimal places using a Monte Carlo method.

Hint: The basic equation of a circle is x^2 + y^2 = r^2.

## Explanation

Use random sampling (Monte Carlo simulation) to estimate the value of pi accurate to three decimal places.
