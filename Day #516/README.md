## Difficulty

Easy

## Problem Statement

Let's define a "sevenish" number to be one which is either a power of `7`, or the sum of unique powers of `7`. The first few sevenish numbers are `1`, `7`, `8`, `49`, and so on. Create an algorithm to find the `n`th sevenish number.

## Example

### Input
```
The first few sevenish numbers: 1, 7, 8, 49, ...
```
### Output
```
The nth sevenish number
```

## Explanation

Find the nth sevenish number by treating n's binary representation as a selection of which unique powers of 7 to sum.
