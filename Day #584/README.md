# Day 584

## Difficulty

Easy

## Problem Statement

Given a string with repeated characters, rearrange the string so that no two adjacent characters are the same. If this is not possible, return `None`.

## Example

### Input
```
"aaabbc"
```
### Output
```
"ababac"
```

## Explanation

Reorder a string's characters so no two neighboring characters match (e.g. using a greedy/heap approach by most frequent character), returning None when impossible like "aaab".
