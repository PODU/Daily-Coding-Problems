# Day 974

## Difficulty

Hard

## Problem Statement

Given a string consisting of parentheses, single digits, and positive and negative signs, convert the string into a mathematical expression to obtain the answer.

Don't use `eval` or a similar built-in parser.

## Example

### Input
```
'-1 + (2 + 3)'
```
### Output
```
4
```

## Explanation

Parse and evaluate a mathematical expression string containing parentheses, single digits, and plus/minus signs without using a built-in evaluator.
