# Day 1269

## Difficulty

Medium

## Problem Statement

Given an N by N matrix, rotate it by 90 degrees clockwise.

For example, given the following matrix:

```
[[1, 2, 3],
 [4, 5, 6],
 [7, 8, 9]]
```

you should return:

```
[[7, 4, 1],
 [8, 5, 2],
 [9, 6, 3]]
```

Follow-up: What if you couldn't use any extra space?

## Example

### Input
```
[[1, 2, 3],
 [4, 5, 6],
 [7, 8, 9]]
```
### Output
```
[[7, 4, 1],
 [8, 5, 2],
 [9, 6, 3]]
```

## Explanation

Rotate an N by N matrix 90 degrees clockwise, ideally in place without extra space.
