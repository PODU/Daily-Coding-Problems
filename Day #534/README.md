# Day 534

## Difficulty

Easy

## Problem Statement

Given a string, determine whether any permutation of it is a palindrome.

## Example

### Input
```
carrace
daily
```
### Output
```
true
false
```

## Explanation

Check whether some rearrangement of the string is a palindrome, which holds when at most one character has an odd count. "carrace" returns true, "daily" returns false.
