# Day 788

## Difficulty

Easy

## Problem Statement

Write a program that checks whether an integer is a palindrome. For example, `121` is a palindrome, as well as `888`. `678` is not a palindrome. Do not convert the integer into a string.

## Example

### Input
```
121
```
### Output
```
true
```

## Explanation

Determine whether an integer reads the same forwards and backwards without converting it to a string; 121 and 888 are palindromes, 678 is not.
