# Day 785

## Difficulty

Easy

## Problem Statement

Given a list of integers, return the largest product that can be made by multiplying any three integers.

You can assume the list has at least three integers.

## Example

### Input
```
[-10, -10, 5, 2]
```
### Output
```
500
```

## Explanation

Return the maximum product of any three integers; for [-10, -10, 5, 2] it is 500 from -10 * -10 * 5.
