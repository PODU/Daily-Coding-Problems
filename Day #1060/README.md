# Day 1060

## Difficulty

Easy

## Problem Statement

Given a function `f`, and `N` return a debounced `f` of `N` milliseconds.

That is, as long as the debounced `f` continues to be invoked, `f` itself will not be called for `N` milliseconds.

## Explanation

Implement a debounce wrapper that delays calling the given function until it has stopped being invoked for N milliseconds.
