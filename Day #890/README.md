# Day 890

## Difficulty

Medium

## Problem Statement

There is an N by M matrix of zeroes. Given N and M, write a function to count the number of ways of starting at the top-left corner and getting to the bottom-right corner. You can only move right or down.

## Example

### Input
```
2 by 2 matrix
5 by 5 matrix
```
### Output
```
2
70
```

## Explanation

Count the number of lattice paths from top-left to bottom-right of an N by M grid moving only right or down.
