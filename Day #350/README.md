# Day 350

## Difficulty

Medium

## Problem Statement

Write a program that determines the smallest number of perfect squares that sum up to N.

## Example

### Input
```
N = 4
N = 17
N = 18
```
### Output
```
1 (4)
2 (16 + 1)
2 (9 + 9)
```

## Explanation

Given an integer N, find the minimum count of perfect squares that add up to N.
