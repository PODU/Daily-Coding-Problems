# Day 1062

## Difficulty

Easy

## Problem Statement

Given the head of a singly linked list, swap every two nodes and return its head.

## Example

### Input
```
1 -> 2 -> 3 -> 4
```
### Output
```
2 -> 1 -> 4 -> 3
```

## Explanation

Swap each adjacent pair of nodes in a singly linked list and return the new head.
