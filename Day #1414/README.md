# Day 1414

## Difficulty

Easy

## Problem Statement

Determine whether there exists a one-to-one character mapping from one string s1 to another s2.

## Example

### Input
```
s1 = abc, s2 = bcd
s1 = foo, s2 = bar
```
### Output
```
true
false
```

## Explanation

Given two strings, decide if a consistent one-to-one mapping of characters from s1 to s2 exists (e.g. abc to bcd works, but foo to bar fails because o would need to map to two different characters).
