# Day 1151

## Difficulty

Easy

## Problem Statement

Determine whether a doubly linked list is a palindrome. What if it's singly linked?

## Example

### Input
```
1 -> 4 -> 3 -> 4 -> 1
1 -> 4
```
### Output
```
True
False
```

## Explanation

Check whether a linked list reads the same forwards and backwards, considering both the doubly linked and singly linked cases.
