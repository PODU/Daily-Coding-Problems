# Day 1055

## Difficulty

Medium

## Problem Statement

Given a node in a binary search tree, return the next bigger element, also known as the inorder successor.

You can assume each node has a `parent` pointer.

## Example

### Input
```
   10
  /  \
 5    30
     /  \
   22    35

node = 22
```
### Output
```
30
```

For example, the inorder successor of 22 is 30.

## Explanation

Given a node (with a parent pointer) in a binary search tree, find its inorder successor, the next-largest value in the tree.
