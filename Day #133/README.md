# Day 133

## Difficulty

Medium

## Problem Statement

Given a node in a binary search tree, return the next bigger element, also known as the inorder successor.

You can assume each node has a `parent` pointer.

## Example

### Input
```
   10
  /  \
 5    30
     /  \
   22    35

node 22
```
### Output
```
30
```

## Explanation

Given a node (with a parent pointer) in a BST, return its inorder successor, i.e. the next-larger value.
