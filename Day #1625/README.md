# Day 1625

## Difficulty

Medium

## Problem Statement

Given a node in a binary search tree, return the next bigger element, also known as the inorder successor.

You can assume each node has a `parent` pointer.

## Example

### Input
```
   10
  /  \
 5    30
     /  \
   22    35
```
### Output
```
The inorder successor of 22 is 30.
```

## Explanation

Find the inorder successor (next larger value) of a given node in a binary search tree, using parent pointers.
