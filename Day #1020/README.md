# Day 1020

## Difficulty

Easy

## Problem Statement

Given the root of a binary search tree, and a target `K`, return two nodes in the tree whose sum equals `K`.

## Example

### Input
```
    10
   /   \
 5      15
       /  \
     11    15

K = 20
```
### Output
```
5 and 15
```

## Explanation

Find two nodes in a binary search tree whose values add up to a given target K.
