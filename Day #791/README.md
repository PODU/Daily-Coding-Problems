# Day 791

## Difficulty

Medium

## Problem Statement

Write a function, `throw_dice(N, faces, total)`, that determines how many ways it is possible to throw `N` dice with some number of faces each to get a specific total.

## Example

### Input
```
throw_dice(3, 6, 7)
```
### Output
```
15
```

## Explanation

Count the number of ways to roll N dice (each with the given number of faces) that sum to the target total; throw_dice(3, 6, 7) is 15.
