# Day 900

## Difficulty

Medium

## Problem Statement

Given an array of numbers, find the maximum sum of any contiguous subarray of the array.

Do this in O(N) time.

## Example

### Input
```
[34, -50, 42, 14, -5, 86]
[-5, -1, -8, -9]
```
### Output
```
137
0
```

## Explanation

Find the maximum sum of any contiguous subarray in O(N) time (Kadane's algorithm); an empty subarray summing to 0 is allowed.
