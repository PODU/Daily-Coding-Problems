# Day 1783

## Difficulty

Medium

## Problem Statement

You are given a 2-d matrix where each cell consists of either /, \, or an empty space. Write an algorithm that determines into how many regions the slashes divide the space.

## Example

### Input
```
\    /
 \  /
  \/
```
### Output
```
3
```

## Explanation

Count the number of regions a grid is divided into by forward and backward slashes, treating the matrix edges as boundaries.
