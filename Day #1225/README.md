# Day 1225

## Difficulty

Medium

## Problem Statement

Given a string of parentheses, write a function to compute the minimum number of parentheses to be removed to make the string valid (i.e. each open parenthesis is eventually closed).

## Example

### Input
```
"()())()"
")("
```
### Output
```
1
2
```

## Explanation

Compute the minimum number of parentheses to delete from a string so that the remaining parentheses are balanced and valid.
