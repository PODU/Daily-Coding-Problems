# Day 1266

## Difficulty

Medium

## Problem Statement

Given a list of numbers, create an algorithm that arranges them in order to form the largest possible integer.

## Example

### Input
```
[10, 7, 76, 415]
```
### Output
```
77641510
```

## Explanation

Arrange a list of numbers so that their concatenation forms the largest possible integer.
