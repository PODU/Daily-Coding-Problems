# Day 988

## Difficulty

Medium

## Problem Statement

Given a positive integer `n`, find the smallest number of squared integers which sum to `n`.

## Example

### Input
```
n = 13
n = 27
```
### Output
```
2 since 13 = 3^2 + 2^2 = 9 + 4
3 since 27 = 3^2 + 3^2 + 3^2 = 9 + 9 + 9
```

## Explanation

Find the minimum count of perfect square numbers that sum to a given positive integer n.
