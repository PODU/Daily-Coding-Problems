# Day 556

## Difficulty

Medium

## Problem Statement

Given an array of integers, write a function to determine whether the array could become non-decreasing by modifying at most 1 element.

## Example

### Input
```
[10, 5, 7]
[10, 5, 1]
```
### Output
```
true
false
```

## Explanation

Determine if the array can be made non-decreasing by changing at most one element. [10, 5, 7] returns true (modify 10 into 1), while [10, 5, 1] returns false.
