# Day 106

## Difficulty

Medium

## Problem Statement

Given an integer list where each number represents the number of hops you can make, determine whether you can reach to the last index starting at index 0.

## Example

### Input
```
[2, 0, 1, 0]
[1, 1, 0, 1]
```
### Output
```
True
False
```

## Explanation

Decide whether you can jump from index 0 to the last index given the hop counts.
