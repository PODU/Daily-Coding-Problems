# Day 1243

## Difficulty

Easy

## Problem Statement

Given two strings A and B, return whether or not A can be shifted some number of times to get B.

## Example

### Input
```
A = "abcde", B = "cdeab"
A = "abc", B = "acb"
```
### Output
```
true
false
```

## Explanation

Determine whether string B is a rotation (cyclic shift) of string A.
