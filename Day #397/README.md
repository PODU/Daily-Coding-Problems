# Day 397

## Difficulty

Medium

## Problem Statement

You are given a list of jobs to be done, where each job is represented by a start time and end time. Two jobs are compatible if they don't overlap. Find the largest subset of compatible jobs.

There is no guarantee that jobs will be sorted.

## Example

### Input
```
[(0, 6),
(1, 4),
(3, 5),
(3, 8),
(4, 7),
(5, 9),
(6, 10),
(8, 11)]
```
### Output
```
[(1, 4),
(4, 7),
(8, 11)]
```

## Explanation

Select the largest subset of jobs (start/end intervals) such that none of them overlap.
