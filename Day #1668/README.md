# Day 1668

## Difficulty

Easy

## Problem Statement

A number is considered perfect if its digits sum up to exactly 10.

Given a positive integer `n`, return the `n`-th perfect number.

## Example

### Input
```
n = 1
n = 2
```
### Output
```
19
28
```

## Explanation

A "perfect" number here is one whose digits sum to 10; given n, return the n-th such number in increasing order.
