# Day 931

## Difficulty

Easy

## Problem Statement

Given `n` numbers, find the greatest common denominator between them.

For example, given the numbers `[42, 56, 14]`, return `14`.

## Example

### Input
```
[42, 56, 14]
```
### Output
```
14
```

## Explanation

Compute the greatest common divisor of a list of n numbers by folding the pairwise GCD across them.
