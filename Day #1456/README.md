# Day 1456

## Difficulty

Easy

## Problem Statement

Given a string, return the first recurring character in it, or null if there is no recurring character.

## Example

### Input
```
"acbbac"
"abcdef"
```
### Output
```
"b"
null
```

## Explanation

Return the first character that appears more than once (the first whose duplicate occurrence is earliest), or null if all characters are unique.
