# Day 1036

## Difficulty

Medium

## Problem Statement

Given the sequence of keys visited by a postorder traversal of a binary search tree, reconstruct the tree.

## Example

### Input
```
2, 4, 3, 8, 7, 5
```
### Output
```
    5
   / \
  3   7
 / \   \
2   4   8
```

## Explanation

Rebuild a binary search tree from the sequence of keys produced by its postorder traversal.
