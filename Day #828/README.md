# Day 828

## Difficulty

Medium

## Problem Statement

Write a program to determine how many distinct ways there are to create a max heap from a list of `N` given integers.

## Example

### Input
```
N = 3, integers = [1, 2, 3]
```
### Output
```
2
```

## Explanation

Count the distinct max heaps that can be built from N given integers. For [1, 2, 3] there are two valid arrangements (3 over 1,2 and 3 over 2,1).
