# Day 393

## Difficulty

Medium

## Problem Statement

Given an array of integers, return the largest range, inclusive, of integers that are all included in the array.

## Example

### Input
```
[9, 6, 1, 3, 8, 10, 12, 11]
```
### Output
```
(8, 12)
```

## Explanation

Find the largest inclusive range of consecutive integers all present in the array; for the example 8, 9, 10, 11, and 12 are all in the array.
