# Day 1574

## Difficulty

Medium

## Problem Statement

Given a list of words, return the shortest unique prefix of each word. For example, given the list:

* dog
* cat
* apple
* apricot
* fish

Return the list:

* d
* c
* app
* apr
* f

## Example

### Input
```
dog
cat
apple
apricot
fish
```
### Output
```
d
c
app
apr
f
```

## Explanation

Given a list of words, return for each word the shortest prefix that uniquely identifies it among all the words.
