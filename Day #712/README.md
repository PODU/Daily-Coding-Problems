# Day 712

## Difficulty

Easy

## Problem Statement

Given a string of round, curly, and square open and closing brackets, return whether the brackets are balanced (well-formed).

## Example

### Input
```
"([])[]({})"
"([)]"
"((()"
```
### Output
```
true
false
false
```

## Explanation

Determine whether a string of round, curly, and square brackets is properly nested and balanced.
