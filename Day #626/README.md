# Day 626

## Difficulty

Easy

## Problem Statement

Given a list of integers, return the largest product that can be made by multiplying any three integers.

You can assume the list has at least three integers.

## Example

### Input
```
[-10, -10, 5, 2]
```
### Output
```
500
```

## Explanation

Find the maximum product obtainable from any three integers in the list. In the example, 500 comes from -10 * -10 * 5.
