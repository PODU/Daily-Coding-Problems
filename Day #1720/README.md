# Day 1720

## Difficulty

Medium

## Problem Statement

Write a program that determines the smallest number of perfect squares that sum up to N.

## Example

### Input
```
N = 4
N = 17
N = 18
```
### Output
```
1 (4)
2 (16 + 1)
2 (9 + 9)
```

## Explanation

Find the minimum number of perfect squares that sum to a given integer N.
