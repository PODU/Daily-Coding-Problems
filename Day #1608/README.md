# Day 1608

## Difficulty

Medium

## Problem Statement

Write a program to determine how many distinct ways there are to create a max heap from a list of `N` given integers.

## Example

### Input
```
N = 3, integers = [1, 2, 3]
```
### Output
```
2

  3      3
 / \    / \
1   2  2   1
```

## Explanation

Count the number of distinct max heaps that can be formed from N given integers.
