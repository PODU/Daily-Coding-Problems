# Day 1285

## Difficulty

Medium

## Problem Statement

Given a string and a number of lines k, print the string in zigzag form. In zigzag, characters are printed out diagonally from top left to bottom right until reaching the kth line, then back up to top right, and so on.

## Example

### Input
```
"thisisazigzag", k = 4
```
### Output
```
t     a     g
 h   s z   a
  i i   i z
   s     g
```

## Explanation

Print the characters of a string in a zigzag pattern across k lines, descending diagonally then ascending and repeating.
