# Day 1597

## Difficulty

Medium

## Problem Statement

Given pre-order and in-order traversals of a binary tree, write a function to reconstruct the tree.

## Example

### Input
```
Preorder traversal: [a, b, d, e, c, f, g]
Inorder traversal:  [d, b, e, a, f, c, g]
```
### Output
```
    a
   / \
  b   c
 / \ / \
d  e f  g
```

## Explanation

Reconstruct the original binary tree given its pre-order and in-order traversal sequences.
