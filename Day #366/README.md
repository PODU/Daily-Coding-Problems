# Day 366

## Difficulty

Medium

## Problem Statement

Given a string `s`, rearrange the characters so that any two adjacent characters are not the same. If this is not possible, return null.

## Example

### Input
```
s = yyz
s = yyy
```
### Output
```
yzy
null
```

## Explanation

Reorder the characters of a string so no two adjacent characters are equal, or return null if no such arrangement exists.
