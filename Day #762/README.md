# Day 762

## Difficulty

Hard

## Problem Statement

Find an efficient algorithm to find the smallest distance (measured in number of words) between any two given words in a string.

## Example

### Input
```
words "hello" and "world", text content "dog cat hello cat dog dog hello cat world"
```
### Output
```
1
```

## Explanation

Return 1 because there's only one word "cat" in between the two words. Compute the minimum word-gap between two target words in a text.
