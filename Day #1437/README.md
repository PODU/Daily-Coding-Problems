# Day 1437

## Difficulty

Hard

## Problem Statement

Given an array of numbers, find the length of the longest increasing subsequence in the array. The subsequence does not necessarily have to be contiguous.

## Example

### Input
```
[0, 8, 4, 12, 2, 10, 6, 14, 1, 9, 5, 13, 3, 11, 7, 15]
```
### Output
```
6
```

## Explanation

Find the length of the longest strictly increasing (not necessarily contiguous) subsequence of an array. For the example, one such subsequence is 0, 2, 6, 9, 11, 15 with length 6.
