# Day 1436

## Difficulty

Easy

## Problem Statement

Given an array of elements, return the length of the longest subarray where all its elements are distinct.

## Example

### Input
```
[5, 1, 3, 5, 2, 3, 4, 1]
```
### Output
```
5
```

## Explanation

Find the length of the longest contiguous subarray containing no repeated elements. For the example, the longest such subarray is [5, 2, 3, 4, 1] with length 5.
