# Day 1848

## Difficulty

Hard

## Problem Statement

Write a program that computes the length of the longest common subsequence of three given strings.

## Example

### Input
```
"epidemiologist", "refrigeration", "supercalifragilisticexpialodocious"
```
### Output
```
5
```

## Explanation

Compute the length of the longest common subsequence shared by three strings (e.g. "eieio" has length 5).
