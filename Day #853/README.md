# Day 853

## Difficulty

Hard

## Problem Statement

Find an efficient algorithm to find the smallest distance (measured in number of words) between any two given words in a string.

## Example

### Input
```
words = "hello", "world"
text = "dog cat hello cat dog dog hello cat world"
```
### Output
```
1
```

## Explanation

Given two words and a text, find the minimum number of words separating any occurrence of one word from any occurrence of the other.
