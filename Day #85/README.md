# Day 85

## Difficulty

Medium

## Problem Statement

Given three 32-bit integers x, y, and b, return x if b is 1 and y if b is 0, using only mathematical or bit operations. You can assume b can only be 1 or 0.

## Explanation

Select between two integers based on a boolean flag b, using only arithmetic or bit manipulation rather than conditional branching.
