# Day 934

## Difficulty

Easy

## Problem Statement

Given a string, return the first recurring character in it, or null if there is no recurring character.

## Example

### Input
```
"acbbac"
"abcdef"
```
### Output
```
"b"
null
```

## Explanation

Return the first character in a string that appears more than once, or null if every character is unique.
