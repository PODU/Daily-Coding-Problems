# Day 1144

## Difficulty

Easy

## Problem Statement

Given integers `M` and `N`, write a program that counts how many positive integer pairs `(a, b)` satisfy the following conditions:

* `a + b = M`
* `a XOR b = N`

## Explanation

Given integers M and N, count the number of positive integer pairs (a, b) such that a + b = M and a XOR b = N.
