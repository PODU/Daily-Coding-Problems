# Day 339

## Difficulty

Easy

## Problem Statement

Given an array of numbers and a number `k`, determine if there are three entries in the array which add up to the specified number `k`.

## Example

### Input
```
[20, 303, 3, 4, 25] and k = 49
```
### Output
```
true as 20 + 4 + 25 = 49.
```

## Explanation

Given an array and a target k, determine whether any three elements of the array sum to k.
