# Day 1416

## Difficulty

Hard

## Problem Statement

Given a string consisting of parentheses, single digits, and positive and
negative signs, convert the string into a mathematical expression to obtain the
answer.

Don't use eval or a similar built-in parser.

## Example

### Input
```
'-1 + (2 + 3)'
```
### Output
```
4
```

## Explanation

Parse and evaluate a string of parentheses, single digits, and plus/minus signs into its numeric result without using eval or a built-in parser.
