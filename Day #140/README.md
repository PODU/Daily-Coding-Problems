# Day 140

## Difficulty

Medium

## Problem Statement

Given an array of integers in which two elements appear exactly once and all other elements appear exactly twice, find the two elements that appear only once.

Follow-up: Can you do this in linear time and constant space?

## Example

### Input
```
[2, 4, 6, 8, 10, 2, 6, 10]
```
### Output
```
4 and 8
```

## Explanation

In an array where exactly two values appear once and all others appear twice, identify the two unique values, ideally in O(n) time and O(1) space.
