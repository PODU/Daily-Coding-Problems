# Day 1027

## Difficulty

Medium

## Problem Statement

Given an unsorted array of integers, find the length of the longest consecutive elements sequence.

Your algorithm should run in O(n) complexity.

## Example

### Input
```
[100, 4, 200, 1, 3, 2]
```
### Output
```
4
```

## Explanation

Find the length of the longest run of consecutive integers present in an unsorted array, running in O(n) time.
