# Day 399

## Difficulty

Hard

## Problem Statement

Given a list of strictly positive integers, partition the list into 3 contiguous partitions which each sum up to the same value. If not possible, return null.

## Example

### Input
```
[3, 5, 8, 0, 8]
```
### Output
```
[[3, 5],
 [8, 0],
 [8]]
```

## Explanation

Split the list into three contiguous parts that each have the same sum (8 in the example), or return null if impossible.
