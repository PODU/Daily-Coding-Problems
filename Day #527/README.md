# Day 527

## Difficulty

Medium

## Problem Statement

Write a program to determine how many distinct ways there are to create a max heap from a list of `N` given integers.

## Example

### Input
```
N = 3, integers = [1, 2, 3]
```
### Output
```
2
```

## Explanation

Count the distinct max heaps that can be formed from N integers, using combinatorics over the heap's tree structure (the example [1, 2, 3] yields two arrangements).
