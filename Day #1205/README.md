## Difficulty

Easy

## Problem Statement

Let's represent an integer in a linked list format by having each node represent a digit in the number. The nodes make up the number in reversed order.

For example, the following linked list:

```
1 -> 2 -> 3 -> 4 -> 5
```

is the number 54321.

Given two linked lists in this format, return their sum in the same linked list format.

## Example

### Input
```
9 -> 9
5 -> 2
```
### Output
```
4 -> 2 -> 1
```

## Explanation

Add two numbers that are each represented as a linked list of digits stored in reversed order, returning the sum in the same reversed linked list format. For 99 + 25 = 124, the result is 4 -> 2 -> 1.
