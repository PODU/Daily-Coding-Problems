# Day 474

## Difficulty

Hard

## Problem Statement

Find the minimum number of coins required to make `n` cents.

You can use standard American denominations, that is, 1¢, 5¢, 10¢, and 25¢.

## Example

### Input
```
n = 16
```
### Output
```
3
```

## Explanation

Find the fewest coins (denominations 1¢, 5¢, 10¢, 25¢) that sum to n cents; for n = 16 the answer is 3 (10¢ + 5¢ + 1¢).
