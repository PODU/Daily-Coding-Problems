# Day 433

## Difficulty

Medium

## Problem Statement

Given an integer n, find the next biggest integer with the same number of 1-bits on. For example, given the number 6 (`0110` in binary), return 9 (`1001`).

## Example

### Input
```
6 (0110 in binary)
```
### Output
```
9 (1001 in binary)
```

## Explanation

Given an integer, find the next larger integer that has the same number of set (1) bits.
