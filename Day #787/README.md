# Day 787

## Difficulty

Medium

## Problem Statement

Given an integer n, find the next biggest integer with the same number of 1-bits on.

## Example

### Input
```
6 (0110 in binary)
```
### Output
```
9 (1001 in binary)
```

## Explanation

Return the smallest integer larger than n that has the same count of set bits; for 6 (0110) the answer is 9 (1001).
