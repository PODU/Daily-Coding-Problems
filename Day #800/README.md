# Day 800

## Difficulty

Medium

## Problem Statement

Given a linked list, rearrange the node values such that they appear in alternating `low -> high -> low -> high ...` form.

## Example

### Input
```
1 -> 2 -> 3 -> 4 -> 5
```
### Output
```
1 -> 3 -> 2 -> 5 -> 4
```

## Explanation

Reorder the values of a linked list so they zig-zag between smaller and larger values in alternating low/high order.
