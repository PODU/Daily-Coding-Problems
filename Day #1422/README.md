# Day 1422

## Difficulty

Hard

## Problem Statement

Given a list of integers, write a function that returns the largest sum of
non-adjacent numbers. Numbers can be 0 or negative.

Follow-up: Can you do this in O(N) time and constant space?

## Example

### Input
```
[2, 4, 6, 2, 5]  ->  13
[5, 1, 1, 5]     ->  10
```
### Output
```
[2, 4, 6, 2, 5] returns 13 (pick 2, 6, and 5)
[5, 1, 1, 5] returns 10 (pick 5 and 5)
```

## Explanation

Find the largest sum of non-adjacent numbers in a list (values may be 0 or negative), ideally in O(N) time and constant space.
