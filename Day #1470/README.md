# Day 1470

## Difficulty

Medium

## Problem Statement

Given a positive integer `n`, find the smallest number of squared integers which sum to `n`.

## Example

### Input
```
n = 13
n = 27
```
### Output
```
2
3
```

## Explanation

Return the minimum count of perfect squares summing to n (e.g. 13 = 9 + 4 uses 2; 27 = 9 + 9 + 9 uses 3).
