# Day 1640

## Difficulty

Hard

## Problem Statement

You're given a string consisting solely of `(`, `)`, and `*`. `*` can represent either a `(`, `)`, or an empty string. Determine whether the parentheses are balanced.

## Example

### Input
```
(()*
(*)
)*(
```
### Output
```
(()* and (*) are balanced. )*( is not balanced.
```

## Explanation

Determine whether a string of parentheses and wildcards (`*`) can be balanced, where each `*` may stand for `(`, `)`, or an empty string.
