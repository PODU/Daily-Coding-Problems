# Day 994

## Difficulty

Easy

## Problem Statement

Print the nodes in a binary tree level-wise. For example, the following should print 1, 2, 3, 4, 5.

## Example

### Input
```
  1
 / \
2   3
   / \
  4   5
```
### Output
```
1, 2, 3, 4, 5
```

## Explanation

Traverse a binary tree level by level (breadth-first) and print the node values in that order.
