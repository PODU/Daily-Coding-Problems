# Day 485

## Difficulty

Hard

## Problem Statement

Given an integer k and a string s, find the length of the longest substring that contains at most k distinct characters.

## Example

### Input
```
s = "abcba", k = 2
```
### Output
```
3 (the substring "bcb")
```

## Explanation

Find the length of the longest substring of s that uses no more than k distinct characters.
