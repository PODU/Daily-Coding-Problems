# Day 529

## Difficulty

Hard

## Problem Statement

Given a string, split it into as few strings as possible such that each string is a palindrome.

## Example

### Input
```
racecarannakayak
```
### Output
```
["racecar", "anna", "kayak"]
```

## Explanation

Partition a string into the minimum number of substrings where every substring is a palindrome.
