# Day 1281

## Difficulty

Easy

## Problem Statement

Given two rectangles on a 2D graph, return the area of their intersection. If the rectangles don't intersect, return 0.

## Example

### Input
```
{
    "top_left": (1, 4),
    "dimensions": (3, 3) # width, height
}

and

{
    "top_left": (0, 5),
    "dimensions": (4, 3) # width, height
}
```
### Output
```
6
```

## Explanation

Given two axis-aligned rectangles defined by a top-left corner and width/height, compute the area where they overlap, or 0 if they don't.
