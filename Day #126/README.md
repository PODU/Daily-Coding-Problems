# Day 126

## Difficulty

Medium

## Problem Statement

Write a function that rotates a list by k elements. Try solving this without creating a copy of the list. How many swap or move operations do you need?

## Example

### Input
```
[1, 2, 3, 4, 5, 6] rotated by two
```
### Output
```
[3, 4, 5, 6, 1, 2]
```

## Explanation

Rotate a list left by k elements in-place, minimizing the number of swap or move operations.
