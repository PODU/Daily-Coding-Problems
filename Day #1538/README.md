# Day 1538

## Difficulty

Easy

## Problem Statement

Given an array of time intervals (start, end) for classroom lectures (possibly overlapping), find the minimum number of rooms required.

## Example

### Input
```
[(30, 75), (0, 50), (60, 150)]
```
### Output
```
2
```

## Explanation

Determine the minimum number of rooms needed so that no two overlapping lecture intervals share a room.
