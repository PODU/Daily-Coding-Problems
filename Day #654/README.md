# Day 654

## Difficulty

Medium

## Problem Statement

Given a string, find the length of the smallest window that contains every distinct character. Characters may appear more than once in the window.

## Example

### Input
```
"jiujitsu"
```
### Output
```
5
```

## Explanation

Find the shortest contiguous substring that contains all distinct characters of the string; for "jiujitsu" the answer is 5, corresponding to the final five letters.
