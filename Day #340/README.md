# Day 340

## Difficulty

Easy

## Problem Statement

Given a set of points (x, y) on a 2D cartesian plane, find the two closest points.

## Example

### Input
```
[(1, 1), (-1, -1), (3, 4), (6, 1), (-1, -6), (-4, -3)]
```
### Output
```
[(-1, -1), (1, 1)]
```

## Explanation

Given a set of 2D points, find the pair of points with the smallest Euclidean distance between them.
