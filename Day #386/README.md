# Day 386

## Difficulty

Easy

## Problem Statement

Given a string, sort it in decreasing order based on the frequency of characters. If there are multiple possible solutions, return any of them.

## Example

### Input
```
tweet
```
### Output
```
tteew
```

## Explanation

Reorder the characters of a string so that more frequent characters come first; any valid ordering by frequency is acceptable.
