# Day 1236

## Difficulty

Medium

## Problem Statement

Write a function, `throw_dice(N, faces, total)`, that determines how many ways it is possible to throw `N` dice with some number of faces each to get a specific total.

## Example

### Input
```
throw_dice(3, 6, 7)
```
### Output
```
15
```

## Explanation

Count the number of distinct ways to roll N dice, each with a given number of faces, so that the pips sum to a specified total.
