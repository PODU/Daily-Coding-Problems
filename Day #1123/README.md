# Day 1123

## Difficulty

Medium

## Problem Statement

Given a set of distinct positive integers, find the largest subset such that every pair of elements in the subset (i, j) satisfies either i % j = 0 or j % i = 0.

## Example

### Input
```
[3, 5, 10, 20, 21]
[1, 3, 6, 24]
```
### Output
```
[5, 10, 20]
[1, 3, 6, 24]
```

## Explanation

Find the largest divisible subset where every pair of elements has one dividing the other.
