# Day 142

## Difficulty

Hard

## Problem Statement

You're given a string consisting solely of `(`, `)`, and `*`. `*` can represent either a `(`, `)`, or an empty string. Determine whether the parentheses are balanced.

## Example

### Input
```
(()*   and   (*)   and   )*(
```
### Output
```
(()* and (*) are balanced. )*( is not balanced.
```

## Explanation

Decide whether a string of parentheses and wildcards (where * can be '(', ')', or empty) can form a balanced parenthesis sequence.
