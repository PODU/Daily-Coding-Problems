# Day 1328

## Difficulty

Hard

## Problem Statement

Given a string, split it into as few strings as possible such that each string is a palindrome.

## Example

### Input
```
racecarannakayak
abc
```
### Output
```
["racecar", "anna", "kayak"]
["a", "b", "c"]
```

## Explanation

Partition a string into the minimum number of contiguous substrings where every substring is a palindrome.
