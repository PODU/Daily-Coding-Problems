# Day 972

## Difficulty

Easy

## Problem Statement

Given a string with repeated characters, rearrange the string so that no two adjacent characters are the same. If this is not possible, return `None`.

## Example

### Input
```
"aaabbc"
"aaab"
```
### Output
```
"ababac"
None
```

## Explanation

Rearrange the characters of a string so no two neighboring characters match, returning None when it cannot be done.
