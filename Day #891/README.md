# Day 891

## Difficulty

Medium

## Problem Statement

You are given a 2-d matrix where each cell consists of either `/`, `\`, or an empty space. Write an algorithm that determines into how many regions the slashes divide the space.

## Example

### Input
```
\    /
 \  /
  \/
```
### Output
```
3
```

## Explanation

Given a grid of slashes, backslashes, and spaces, count how many distinct regions the slashes carve the space into, treating the matrix edges as boundaries.
