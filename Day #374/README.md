# Day 374

## Difficulty

Hard

## Problem Statement

Given a sorted array `arr` of distinct integers, return the lowest index `i` for which `arr[i] == i`. Return `null` if there is no such index.

## Example

### Input
```
[-5, -3, 2, 3]
```
### Output
```
2
```

## Explanation

In a sorted array of distinct integers, find the smallest index i such that arr[i] equals i, or null if none exists.
