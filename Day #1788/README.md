# Day 1788

## Difficulty

Hard

## Problem Statement

Given a list of points, a central point, and an integer k, find the nearest k points from the central point.

## Example

### Input
```
points = [(0, 0), (5, 4), (3, 1)], central point = (1, 2), k = 2
```
### Output
```
[(0, 0), (3, 1)]
```

## Explanation

Find the k points closest to a given central point.
