# Day 578

## Difficulty

Easy

## Problem Statement

Determine whether there exists a one-to-one character mapping from one string s1 to another s2.

## Example

### Input
```
s1 = abc, s2 = bcd
s1 = foo, s2 = bar
```
### Output
```
true (map a to b, b to c, and c to d)
false (the o cannot map to two characters)
```

## Explanation

Decide if each character of s1 can be consistently mapped to a single character of s2 in a one-to-one fashion.
