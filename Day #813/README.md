# Day 813

## Difficulty

Medium

## Problem Statement

Given an array of integers, write a function to determine whether the array could become non-decreasing by modifying at most 1 element.

## Example

### Input
```
[10, 5, 7]
[10, 5, 1]
```
### Output
```
true
false
```

## Explanation

Decide whether an array can be made non-decreasing by changing at most one element.
