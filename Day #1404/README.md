# Day 1404

## Difficulty

Medium

## Problem Statement

Given a string and a pattern, find the starting indices of all occurrences of the pattern in the string.

## Example

### Input
```
string = "abracadabra", pattern = "abr"
```
### Output
```
[0, 7]
```

## Explanation

Return the list of all starting indices in the string where the given pattern occurs.
