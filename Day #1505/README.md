# Day 1505

## Difficulty

Medium

## Problem Statement

Given an integer list where each number represents the number of hops you can make, determine whether you can reach to the last index starting at index 0.

For example, `[2, 0, 1, 0]` returns `True` while `[1, 1, 0, 1]` returns `False`.

## Example

### Input
```
[2, 0, 1, 0]
```
### Output
```
True
```

## Explanation

Decide whether you can reach the last index of a list, where each value is the maximum number of hops you can take from that position, starting at index 0.
