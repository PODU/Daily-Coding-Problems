# Day 103

## Difficulty

Medium

## Problem Statement

Given a string and a set of characters, return the shortest substring containing all the characters in the set.

If there is no substring containing all the characters in the set, return null.

## Example

### Input
```
string = "figehaeci", set = {a, e, i}
```
### Output
```
"aeci"
```

## Explanation

Find the minimum-length window of the string that contains every character in the given set.
