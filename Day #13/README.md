# Day 13

## Difficulty

Hard

## Problem Statement

Given an integer k and a string s, find the length of the longest substring that contains at most k distinct characters.

## Example

### Input
```
s = "abcba", k = 2
```
### Output
```
3
```

## Explanation

Find the length of the longest substring containing no more than k distinct characters. For s = "abcba" and k = 2, the longest such substring is "bcb" (length 3).
