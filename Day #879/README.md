# Day 879

## Difficulty

Easy

## Problem Statement

Given a list of numbers and a number k, return whether any two numbers from the list add up to k.

Bonus: Can you do this in one pass?

## Example

### Input
```
[10, 15, 3, 7], k = 17
```
### Output
```
true
```

## Explanation

Determine if any two numbers in the list sum to k; the example returns true since 10 + 7 = 17.
