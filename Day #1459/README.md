# Day 1459

## Difficulty

Medium

## Problem Statement

Invert a binary tree.

For example, given the following tree:

```
    a
   / \
  b   c
 / \  /
d   e f
```

should become:

```
  a
 / \
 c  b
 \  / \
  f e  d
```

## Example

### Input
```
    a
   / \
  b   c
 / \  /
d   e f
```
### Output
```
  a
 / \
 c  b
 \  / \
  f e  d
```

## Explanation

Mirror a binary tree by swapping the left and right children at every node.
