# Day 1517

## Difficulty

Easy

## Problem Statement

A unival tree (which stands for "universal value") is a tree where all nodes under it have the same value.

Given the root to a binary tree, count the number of unival subtrees.

## Example

### Input
```
   0
  / \
 1   0
    / \
   1   0
  / \
 1   1
```
### Output
```
5
```

## Explanation

Count how many subtrees in a binary tree are unival, meaning every node in that subtree shares the same value.
