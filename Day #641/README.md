# Day 641

## Difficulty

Easy

## Problem Statement

Given a sorted array, find the smallest positive integer that is not the sum of a subset of the array.

Do this in `O(N)` time.

## Example

### Input
```
[1, 2, 3, 10]
```
### Output
```
7
```

## Explanation

For a sorted array, find the smallest positive integer that cannot be represented as the sum of any subset, in linear time; for [1, 2, 3, 10] the answer is 7.
