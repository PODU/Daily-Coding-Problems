# Day 644

## Difficulty

Easy

## Problem Statement

A unival tree (which stands for "universal value") is a tree where all nodes under it have the same value.

Given the root to a binary tree, count the number of unival subtrees.

## Example

### Input
```
   0
  / \
 1   0
    / \
   1   0
  / \
 1   1
```
### Output
```
5
```

## Explanation

Count subtrees in which every node shares the same value; the example tree has 5 such unival subtrees.
