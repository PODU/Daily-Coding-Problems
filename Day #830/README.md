# Day 830

## Difficulty

Medium

## Problem Statement

Given a list of numbers, create an algorithm that arranges them in order to form the largest possible integer.

## Example

### Input
```
[10, 7, 76, 415]
```
### Output
```
77641510
```

## Explanation

Arrange the numbers in the order that, when concatenated, produces the largest possible integer.
