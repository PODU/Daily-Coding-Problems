# Day 1539

## Difficulty

Easy

## Problem Statement

Given a linked list, remove all consecutive nodes that sum to zero. Print out the remaining nodes.

## Example

### Input
```
3 -> 4 -> -7 -> 5 -> -6 -> 6
```
### Output
```
5
```

## Explanation

Remove every run of consecutive nodes whose values sum to zero (first 3 -> 4 -> -7, then -6 -> 6), leaving the remaining nodes.
