# Day 1605

## Difficulty

Easy

## Problem Statement

You are given a `2 x N` board, and instructed to completely cover the board with the following shapes:

* Dominoes, or `2 x 1` rectangles.
* Trominoes, or `L`-shapes.

Given an integer `N`, determine in how many ways this task is possible.

## Example

### Input
```
N = 4
```
### Output
```
One possible configuration, where A is a domino, and B and C are trominoes:

A B B C
A B C C
```

## Explanation

Given a 2 x N board, count the number of ways to fully tile it using 2 x 1 dominoes and L-shaped trominoes.
