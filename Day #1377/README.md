# Day 1377

## Difficulty

Hard

## Problem Statement

You're given a string consisting solely of `(`, `)`, and `*`. `*` can represent either a `(`, `)`, or an empty string. Determine whether the parentheses are balanced.

## Example

### Input
```
(()*  and  (*)  and  )*(
```
### Output
```
(()* and (*) are balanced. )*( is not balanced.
```

## Explanation

Given a string of parentheses and wildcards (`*` can be `(`, `)`, or empty), determine whether the parentheses can be balanced.
