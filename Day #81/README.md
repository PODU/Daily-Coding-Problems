# Day 81

## Difficulty

Easy

## Problem Statement

Given a mapping of digits to letters (as in a phone number), and a digit string, return all possible letters the number could represent. You can assume each valid number in the mapping is a single digit.

## Example

### Input
```
{"2": ["a", "b", "c"], 3: ["d", "e", "f"], ...}, "23"
```
### Output
```
["ad", "ae", "af", "bd", "be", "bf", "cd", "ce", "cf"]
```

## Explanation

Generate all possible letter combinations that a digit string could represent given a digit-to-letters mapping, like phone number letter combinations.
