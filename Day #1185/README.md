# Day 1185

## Difficulty

Hard

## Problem Statement

Find an efficient algorithm to find the smallest distance (measured in number of words) between any two given words in a string.

## Example

### Input
```
words = "hello", "world"
text = "dog cat hello cat dog dog hello cat world"
```
### Output
```
1
```

## Explanation

Find the minimum number of words separating any occurrences of two given words within a text string.
