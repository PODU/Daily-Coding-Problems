# Day 1345

## Difficulty

Easy

## Problem Statement

Given an array of integers out of order, determine the bounds of the smallest window that must be sorted in order for the entire array to be sorted.

## Example

### Input
```
[3, 7, 5, 6, 9]
```
### Output
```
(1, 3)
```

## Explanation

Find the start and end indices of the smallest subarray that, once sorted, makes the whole array sorted.
